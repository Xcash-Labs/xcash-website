What it is

XCash Labs is revitalizing X-CASH as a modern privacy-focused cryptocurrency built for simplicity, fairness, and long-term sustainability.

Powered by XCash Klassic (XCK), the network combines proven CryptoNote privacy technology with community-driven development and transparent governance. With no premine and no airdrops, XCK returns to the original principles of open and fair cryptocurrency distribution.

Core features

  • Completely rewritten DPoPS consensus, fully updated to the latest Monero core
  • Engineered for speed with consistent 60-second block times
  • Flexible privacy with support for both transparent and private transactions
  • DPOPS staking with VRF-based producer selection for fair, community-governed security
  • DAO Governance - Open governance and community-led development
  • Future Web3 support for cross-chain asset wrapping and minting
  • Monero-Grade Privacy, Energy-Efficient by Design
