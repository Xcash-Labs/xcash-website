Verify Reserve Proof And Transfer To New Chain


Steps to transfer from old Chain

  • (1) Using the existing xcash-wallet-cli to create a reserve proof. For the daemon address use --daemon-address 96.30.193.103:18281:18281
  • (2) Now create the reserve proof for your wallet balance. For the text description use "TRANSFER" or it will not work. Also, if you get an error like "Not enough balance", you may need to make the transfer balance .01 less that the current balance, for example 100.28 becomes 100.27.
  • (3) Use the form below to transfer the funds. If successful the funds will be queued for transfer.
  • (4) Then, using your mnemonic seed for you wallet, create the wallet on the new chain. You can download the wallet from here: https://xcashlabs.org/downloads

Note: this process is not completely automated so expect delays in processing the queue.
